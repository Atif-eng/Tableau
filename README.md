## 🛠️ Technical Tableau Skills Demonstrated

This repository highlights a deep understanding of intermediate and advanced Tableau features, including:

* **Advanced Calculations:** Utilizing Level of Detail (LOD) expressions (`FIXED`, `INCLUDE`, `EXCLUDE`) for multi-granular analysis, complex logical statements, and custom string manipulation.
* **Dynamic Interactivity:** Designing with Context Filters, Set/Parameter Actions, and URL Actions to build highly intuitive, guided user analytics experiences.
* **Data Architecture:** Structuring optimized semantic layers using Tableau Relationships (Noodles), Physical Joins, and Unions across diverse data sources.
* **Design & UX/UI:** Adhering to clean dashboard design principles (F-shape reading patterns, unified color theory, custom containers, and collapsible performance sheets).
* **Performance Optimization:** Maximizing rendering speeds by leveraging extracts, reducing nested LODs, optimizing discrete filters, and managing marks density.

## 📊 Data Engineering & Preparation (Pre-Tableau)

Great dashboards rely on clean data. Before bringing datasets into Tableau, I routinely execute the following preprocessing workflows:
* **SQL / Custom Queries:** Utilizing CTEs, Window Functions, and aggregated views to reduce data granularity before ingestion.
* **Python (Pandas):** Handling missing values, outlier detection, data normalization, and shaping long/wide formats.
* **Tableau Prep Builder:** Building reproducible ETL pipelines for scheduled data refreshes and schema cleanups.
